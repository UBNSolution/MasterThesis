- Aplikace využívá Qt framework, pro multiplatformní práci se soubory a zobrazování uživatelského rozhraní

- Pro běh aplikace je zapotřebí mít Qt knihovny verze 5.9 v systému (na Windowsu je postačující přítomnost DLL vedle aplikace)

- Aplikace byla otestována na systému OS X a Windows 10 v 64-bitových variantách
